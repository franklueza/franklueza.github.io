¿Qué es la Filosofía Mobile First?

En corto: es un concepto pensado para la construcción y diseño de sitios web primero en versión móvil.

Creamos proyectos responsivos y de primer nivel en dispositivos móviles para la web con la biblioteca de componentes front-end más popular, Bootstrap.
Bootstrap es un kit de herramientas de código abierto para desarrollar con HTML, CSS y JS.

¿Qué es el Desarrollo Web?

En corto: los desarrolladores web crean y mantienen sitios web.

El desarrollo web a menudo se divide en dos áreas, desarrollo de front-end y desarrollo de back-end.
Los desarrolladores front-end usan HTML, CSS y JavaScript para construir lo que ve el cliente - lo que se ejecuta en el navegador -.
El HTML significa Hyper Text Markup Language, es el lenguaje para la estructura, cada página web comienza con él. Es la base sobre la cual las otras dos capas CSS y JavaScript - presentación e interactividad, respectivamente - se construyen.
El CSS significa Cascading Style Sheets es un lenguaje de presentación creado para estilizar el contenido
Y finalmente, JavaScript es el motor que impulsa el dinamismo a cada uno de los elementos del sitio.
Los desarrolladores de back-end son responsables de lo que se ejecuta en el servidor, usan lenguajes como Python, Ruby y Java para trabajar con bases de datos para guardar, cambiar y recuperar datos.

¿Qué hace el Diseñador Web?

En corto: aplica el trabajo de los diseñadores y lleva sus diseños a código.

No hay un camino a seguir y no existen procedimientos estándar de la industria. Es inexacto y está lleno de toma de decisiones. Eso significa que con cada nuevo diseño web, lo mejor es asumir lo menos posible y aprender todo lo que pueda.
Las primeras fases del diseño web tienen muchos nombres, pero a veces las personas llaman a este primer bloque la fase de descubrimiento. Porque estamos tratando de encontrar tanta información como podamos, antes de que comencemos a diseñar algo que realmente no se ajusta a las necesidades correctas.
