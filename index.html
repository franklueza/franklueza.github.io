Desarrollo de sitios con WordPress

WordPress

Software libre

Diseñamos y desarrollamos sitios autoadministrables con diseño responsivo, para que tu sitio se vea bien en cualquier dspositivo.

Utilizamos WordPress por su fácil curva de aprendizaje para la administración de contenido y es software libre de código abierto - Open Source -, el cual esta bajo la licencia General Public License (GPLv2 o Posterior) que brinda 4 principales libertades:

  1. Correr el programa para cualquier porpósito
  2. Estudiar cómo funciona el programa y cambiarlo para que haga lo que desee.
  3. Redistribuirlo
  4. Distribuir copias de sus versiones modificadas a otros

WordPress es muy flexible debido a las extemciones llamados Plugins y actualmente a nivel mundial el 25% de las páginas están hechas con este gestor de contenido - CMS - y cuenta con miles de temas gratuitos para comenzar a construir, sin embargo es importante adquirir un tema de pago para permitir un mejor desarrollo del sitio web.
