Lenguajes HTML5 & CSS3.

¿Qué es HTML5?

HTML es un lenguaje de marcado para crear páginas web, sus siglas siginifcan Hyper Text Markup Language. Este tipo de lenguaje utiliza etiquetas (tags) para indicar donde deben de ir los encabezados, imagenes, listas, links, saltos de línea y otros elementos dentro de la página

¿Qué es CSS3?

CSS es una hoja de estilos que se agrega a tu HTML o de manera externa como un archivo css que se enlaza desde tu HTML a través de una etiqueta, sus siglas siginifcan Cascading Style Sheets. Aqui se describe como cada uno de los elementos HTML serán desplegados en pantalla de cualquier dispositivo.
