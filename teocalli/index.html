¿A quién va dirigido?

El Programa Online: Principios de Análisis & Visualización de Datos está diseñado para personas curiosas y motivadas que desean incursionar en el mundo del análisis y visualización de datos, sin importar su experiencia previa. Es ideal para:

  • Profesionales de cualquier industria (marketing, educación, salud, negocios, etc.) que quieran tomar decisiones basadas en datos y comunicar ideas de forma clara.
  • Estudiantes universitarios o recién egresados que busquen adquirir habilidades prácticas en análisis de datos para destacar en el mercado laboral.
  • Emprendedores interesados en entender datos para impulsar sus proyectos o startups.
  • Apasionados por aprender que deseen transformar información en historias visuales impactantes, sin necesidad de conocimientos avanzados en matemáticas o programación.

Plan de estudio

Análisis Estadístico de Datos

Objetivo

Comprender y aplicar conceptos estadísticos fundamentales para analizar datos de manera efectiva, incluyendo medidas de tendencia central, dispersión y distribuciones, utilizando ejemplos del mundo real para interpretar resultados y tomar decisiones informadas, sin requerir conocimientos matemáticos avanzados.

Excel para Análisis de Datos

Objetivo

Dominar las funcionalidades de Microsoft Excel para organizar, analizar y visualizar datos, aprendiendo a usar tablas dinámicas, fórmulas y gráficos para crear reportes claros y profesionales, aplicados en proyectos prácticos que resuelvan problemas reales.

SQL para Análisis de Datos

Objetivo

Adquirir habilidades para consultar y manipular bases de datos relacionales utilizando SQL, extrayendo información relevante y generando insights a través de queries, aplicadas en proyectos que simulen escenarios empresariales, sin necesidad de experiencia previa en programación.

Tableau - Business Intelligence

Objetivo

Diseñar dashboards interactivos y visualizaciones impactantes en Tableau, aprendiendo a conectar datos, crear gráficos dinámicos y narrar historias visuales que comuniquen insights de forma clara, culminando en un proyecto de visualización para un caso de negocio real.

Power BI - Business Intelligence

Objetivo

Utilizar Power BI para transformar datos en informes y dashboards profesionales, dominando herramientas de modelado, visualización y análisis para presentar resultados accionables, aplicados en un proyecto individual que resuelva un desafío de inteligencia de negocios.

R Studio para Análisis Estadístico

Objetivo

Aplicar R Studio para realizar análisis estadísticos y crear visualizaciones de datos, aprendiendo a usar paquetes básicos para explorar datasets y comunicar resultados, mediante proyectos prácticos que no requieran conocimientos previos de programación.

Python para Análisis y Visualización de Datos

Objetivo

Desarrollar habilidades en Python para analizar y visualizar datos, utilizando bibliotecas como Pandas, Matplotlib y Seaborn para procesar datasets y crear gráficos impactantes, culminando en un proyecto final que integre análisis y narrativa visual en un caso real.
