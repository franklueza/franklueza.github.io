Acerca del taller

Crea y administra una página web utilizando Wordpress, el CMS más popular del mundo. Aprende a utilizar esta herramienta para diseñar y administrar una página web profesional para tu marca personal.

El taller está dirigido a estudiantes, profesionales y emprendedores, apasionados por emprender y hacer uso de las nuevas tecnologías.

Tener una página web que te permita enseñar quien eres y demostrar todo aquello que te hace especial.
